# Contributing to MCModSec

First off, thanks for checking out the project! If you want to help make downloading Minecraft mods a little less sketchy, you are in the perfect place. Whether you are finding bugs, suggesting new features, or actually writing Python code, all help is super appreciated.

This is a very chill, beginner-friendly repository. If this is your first time contributing to an open-source project, do not stress about being perfect.

## Ways You Can Help

1. Report a Bug
If the scanner randomly crashes on a specific .jar file or spits out completely wrong metadata, please let me know. Head over to the "Issues" tab and open a new bug report. Just drop in the error message and, if possible, tell me which mod caused it so I can test it on my end.

2. Suggest a Feature
Do you know a better regular expression to hunt for malware? Want to add support for a brand-new mod loader? Open an Issue.

3. Write Some Code
If you want to tackle one of the open issues or fix a bug yourself, you are more than welcome to submit a Pull Request
